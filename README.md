# C Programming Tasks

This repository contains basic C programming implementations as part of academic practice.

## Programs Included

### 1. Basic Calculator
A menu-driven calculator program written in C that performs basic arithmetic operations.

**Operations:**
- Addition
- Subtraction
- Multiplication
- Division

### 2. Matrix Operations
A C program to perform basic matrix operations using 2D arrays.

**Operations:**
- Matrix Addition
- Matrix Multiplication
- Matrix Transpose

## Technologies Used
- C Programming Language
- Standard I/O Library

## How to Run
1. Compile the required `.c` file using any C compiler.
2. Run the generated executable.
3. Follow the on-screen menu instructions.
